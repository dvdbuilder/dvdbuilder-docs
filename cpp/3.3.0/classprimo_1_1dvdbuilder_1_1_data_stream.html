Detailed Description

The DataStream interface can be used to implement a data source for VOB(Video Object) data.

To use this interface you must also provide implementation of the InputDataStreamFactory interface.

1.10

virtual uint64_t size ( )
pure virtual

Returns the size of the stream object.

In some cases the size() method may be called without calling the Open method first. This method should return a valid size even when the stream is closed.

Returns
The number of bytes in the stream.
When the stream size is unknown return 0. DVDBuilder will stop reading from the stream when the DataStream::read method returns 0.
To indicate an error condition return uint64_t(-1).
