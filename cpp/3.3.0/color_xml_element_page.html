<color> Element

Description

Specifies a color from the sub-picture color palette.

<color
   index="num"
   value="#FFFFFF"/>
Required Attributes
Attribute Option Description
index 1 to 16 Index of the color in the sub-picture palette.
value This is an RGB color value expressed in hexadecimal format ("#FFFFFF"). Color value.
